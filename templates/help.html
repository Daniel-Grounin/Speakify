Help / Instructions

Welcome to Speakify! This page will guide you on how to use the various features of the application.

1. Uploading a PDF

To start, simply go to the "Upload PDF" page, select the PDF file from your computer, and click the "Upload" button. The application will process the file and extract the text content.

2. Viewing Uploaded PDFs

After uploading a PDF, you can view all your uploaded files on the "View Uploaded PDFs" page. This page will show the text extracted from each PDF and allow you to listen to the audio version.

3. Converting PDFs to Audio

If you want to convert all your uploaded PDFs to audio in one go, use the "Convert All PDFs to Audio" button. The application will generate an audio file for each PDF, which you can listen to later.

4. Deleting PDFs

If you want to remove all the PDFs you have uploaded, you can click the "Delete All PDFs" button. This will permanently delete all the files and their associated audio.

5. Preferences

On the "Preferences" page, you can customize settings such as the language for the audio conversion. Make sure to save your preferences before leaving the page.

6. Getting Help

If you need further assistance or have any questions, feel free to contact our support team. We're here to help you make the most of Speakify!
